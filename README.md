# Mail Sense (✦ Mail Sense)

AI-powered Screen Translation, Summarization, and Spell Check Tool.

## 🔮 Features
- **✨ Wingardium Leviosa**: Select any region on your screen for monitoring.
- **🔮 Asurabalbalta**: Summarize extracted text from the selected region using Gemini AI.
- **💡 Lumos**: High-quality English to Korean translation using Gemini Vision.
- **🪄 Abracadabra**: Check spelling and grammar for both Korean and English texts.
- **🤖 Smart AI Fallback**: Automatically switches between Gemini models (`2.5-flash`, `2.0-flash`, `1.5-flash`, `1.5-pro`) to handle API quota limits without interruption.

## 🛠 Prerequisites
- **Python 3.8+**
- **Tesseract OCR**: Must be installed on the system.
- **Google Gemini API Key**: [Get one here](https://aistudio.google.com/).

## 🚀 Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python mail_sense.pyw
   ```
3. Set your API key in the settings (Gear icon ⚙).
